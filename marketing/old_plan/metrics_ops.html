Metrics, analytics, and operations (general sketch)

North-star metrics (pick 1–2)

  • Activated users per week (e.g., follow + start/save within first session)
  • Activated creators per month (publish N items + first audience outcome)

Key metrics by stage

  • Acquisition: SEO clicks, social clicks, referral sources, cost (if paid)
  • Activation: visit → signup, signup → follow, follow → start, start → return
  • Retention: D1/D7/D30, “continue” usage, notification/email-driven returns
  • Supply: creators onboarded, creators active, content published per niche
  • Quality: completion rates, saves, likes, reports, content satisfaction proxy

Analytics basics (implementation-level placeholders)

  • event taxonomy: view, follow, start, save, continue, publish, share
  • UTM discipline for all campaigns; short links per creator campaign
  • dashboards: acquisition → activation → retention; creator funnel; niche performance

Ops cadence

  • Weekly: review top niches, top landing pages, creator activation list, content gaps
  • Monthly: add/remove niches, update topic map, partner pipeline review

Experimentation

  • run small landing page tests (headline, CTA, proof points)
  • measure by cohort retention and activation (not vanity clicks)
